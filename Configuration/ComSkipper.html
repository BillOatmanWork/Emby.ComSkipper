Commercial Skipper

Automatically skip commercials during recording playback if it has an EDL file created.
Disable the on-screen message when a commercial is skipped.
Show the commercial duration, in seconds, in the on-screen message when a commercial is skipped.
Enabling this allows you to watch what is being recorded and have it skip commercials. But at the cost of using more system resources. See PDF file for implementation details.
This plugin is completely free.
